# Webapp

React + Vite glass dashboard on **10784**, proxying API to **10783**.

## Pages

| Route | Purpose |
|-------|---------|
| `/dashboard` | Health, stats, liveness |
| `/notebooks` | List, create, sources, query |
| `/studio` | Artifact status |
| `/pipeline` | Fleet research pipeline |
| `/tools` | MCP capabilities |
| `/settings` | `nlm doctor` output |
| `/apps` | Fleet hub links |
| `/swagger` | OpenAPI iframe |

Start: `.\start.ps1` or `.\web_sota\start.ps1`
